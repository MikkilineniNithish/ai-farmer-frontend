# 🌾 AgriSmart AI Platform — Frontend

React chat UI for the AI Farmer Assistant.

## 🚀 Live Demo
👉 https://ai-farmer-frontend.vercel.app

## ✨ Features
- 💬 Chat with AI about your crops and weather
- 🌿 Upload plant photo for disease diagnosis
- 📜 View your conversation history
- 🔐 Login and Signup with email

## 🛠️ Tech Stack
- React.js
- Supabase Auth
- Vercel deployment

## 📦 Setup Locally

git clone https://github.com/MikkilineniNithish/ai-farmer-frontend
cd ai-farmer-frontend
npm install
npm start
